# App Store and Commercial IP Checklist

Use this checklist before publishing Moonlit Firefly Bloom commercially on web, iOS, Android, or other platforms.

App stores may not request every license during normal review, but documentation should be ready if questions or disputes arise.

## Audio

- [ ] All Suno sounds are logged in `AUDIO_ASSET_LOG.md`.
- [ ] Suno Pro/Premier proof is saved for the generation dates.
- [ ] Original WAV exports are saved outside the repo.
- [ ] Compressed production versions are saved and mapped to original WAV exports.
- [ ] Prompt text, generation links, or generation IDs are saved where available.
- [ ] No sounds intentionally copy famous songs, films, games, brands, characters, or distinctive sound marks.

## Visual

- [ ] All ChatGPT/OpenAI-generated visual assets are logged in `VISUAL_ASSET_LOG.md`.
- [ ] Canva background-removal, editing, and export usage is logged.
- [ ] Canva stock content is logged separately if it is ever used.
- [ ] Tool/source and edit history are documented, distinguishing original generation from later editing/export.
- [ ] Original generated files are saved.
- [ ] Final edited/exported PNGs are saved.
- [ ] Prompts and generation records are kept where practical.
- [ ] No copyrighted characters, logos, or brands are used without permission.
- [ ] No final marketing or asset prompts rely on "in the style of" living artists.

## Code

- [ ] Direct and transitive dependencies are checked.
- [ ] Third-party licenses are reviewed.
- [ ] Required open-source notices are prepared if needed.
- [ ] No intentionally copied proprietary code is included.
- [ ] AI-assisted code changes are reviewed by a human before release.

## Store

- [ ] App name is checked for trademark conflicts.
- [ ] Privacy policy is prepared.
- [ ] Ads or analytics SDK terms are reviewed if added later.
- [ ] Monetization claims are reviewed if added later.
- [ ] Avoid medical claims such as "treats anxiety" or "cures insomnia."
- [ ] Use softer wording such as "soothing," "cozy," "wind-down," and "relaxing."

## AI

- [ ] AI-generated assets are documented.
- [ ] Human selection, editing, and integration are documented.
- [ ] Prompts and generation records are kept where practical.
- [ ] Tool subscriptions, receipts, and commercial-use proof are saved outside the repo.
