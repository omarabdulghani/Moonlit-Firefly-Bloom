# AI Tool Usage Log

This file documents high-level AI tool usage for Moonlit Firefly Bloom. It is practical project documentation and not a legal conclusion about copyright status.

## OpenAI / ChatGPT / Codex

OpenAI tools, including ChatGPT and Codex, were used for game design assistance, prompts, planning, code generation, debugging, documentation, and implementation support.

As between the user and OpenAI, output ownership is governed by OpenAI terms. Generated code and generated documentation may still require awareness of third-party dependencies, open-source licenses, copied snippets, or asset provenance.

## Suno AI

Suno AI was used to generate game sound effects.

All Suno-generated sounds should be tracked in [AUDIO_ASSET_LOG.md](AUDIO_ASSET_LOG.md). Commercial-use support should be documented with proof of the relevant Suno Pro or Premier subscription at the time each sound was generated.

## ChatGPT / OpenAI Image Generation and Canva

ChatGPT / OpenAI image generation was used to generate visual and background assets.

Canva was used as an editing, background-removal, and export tool to create transparent PNGs. Canva should not be treated as the original art source unless Canva stock content, templates, photos, icons, fonts, or elements are used later.

All generated, edited, exported, or renamed visual assets should be tracked in [VISUAL_ASSET_LOG.md](VISUAL_ASSET_LOG.md). Keep original OpenAI-generated files, edited source files, prompt text, generation IDs, Canva export records, receipts, and relevant terms where practical.

If Canva stock content is ever used later, log it separately in [VISUAL_ASSET_LOG.md](VISUAL_ASSET_LOG.md) and note the specific Canva source, license/proof, and final in-game use.

Omar selected the game concept, creative direction, prompts, sounds, visuals, gameplay feel, and final asset choices.

The project integrates and arranges code, audio, visuals, mechanics, timing, controls, interface text, and overall presentation into an original interactive work. Avoid claiming guaranteed copyright registration for raw AI-generated assets without specific legal review.
